# memorial_patterns

This repository holds Patrick’s evolving agentic framework and research collaboration assets.

## Files
- `deep_research_brief.md` — General methodology (Agent-Ready Master Brief).
- `deep_research_instance_brief.md` — Concrete brief for the current milestone (Custom Instructions v2.0+).
- `prompt_persona_library_v1.md` — Shortcuts & persona definitions.
- `project_overview.md` — Roadmap & collaboration rules.
- `status.md` — Live state; must be updated after any contribution.

## Contributor Rules
1) BEFORE: read `project_overview.md` and `status.md`.  
2) AFTER: update `status.md` (and `project_overview.md` if the roadmap/vision changed).  
3) Prefer automated, reproducible improvements (checklists, SOPs, CI, agents).

## Notes
- Agents (e.g., Jules, ChatGPT Agent Mode) are instructed to consult `project_overview.md` and `status.md` before new tasks and to update `status.md` after contributions.
